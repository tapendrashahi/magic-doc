1. Complex Number

Any ordered pair of real numbers (a, b) is known as a complex number.It is denoted by z or w and given by:

z = (a, b) = a + ib

where:
a = Re(z) (real part)
b = Im(z) (imaginary part)

i is the imaginary unit where i² = −1

Examples

z = 2 + 3i
w = −1 + 2i

C = {(a, b) : a, b ∈ R}

Complex Plane

Every real number is a complex number.
x = x + 0i

R ⊂ C (Real numbers are a subset of complex numbers)

2. Operations of Complex Numbers

(1) Equality

Two complex numbers z = (a, b) and w = (c, d) are equal if:

a = c
b = d

(2) Addition

z = (a, b), w = (c, d)

z + w = (a + c, b + d)

Example:
z = (3, 4)
w = (2, 1)
z + w = (5, 5)

(3) Multiplication

z = (a, b), w = (c, d)

z · w = (ac − bd, ad + bc)

Example:
z = (2, 3)
w = (2, 5)
z · w = (−9, 16)

(4) Division

z = a + ib, w = c + id

z / w = [(ac + bd) + i(bc − ad)] / (c² + d²), w ≠ 0

3. Inverse and Identity

  • Additive inverse of z = (a, b) is (−a, −b)
  • Additive identity = (0, 0)
  • Multiplicative identity = (1, 0)
  • Multiplicative inverse of z = a + ib is:
    1 / z = (a / (a² + b²), −b / (a² + b²))

4. Purely Real and Purely Imaginary

Purely real: b = 0
Purely imaginary: a = 0

Every imaginary number is a complex number but every complex number is not imaginary

5. Powers of i

  • i² = −1
  • i³ = −i
  • i⁴ = 1
  • i^(4n+2) = −1
  • i⁻¹ = −i

Example

iⁿ + iⁿ⁺¹ + iⁿ⁺² + iⁿ⁺³ = 0

6. Conjugate of a Complex Number

If z = a + ib, then conjugate z̄ = a − ib

Example:
z = 3 + 4i
z̄ = 3 − 4i

7. Properties of Conjugate

  • z̄̄ = z
  • z + z̄ = 2Re(z)
  • z − z̄ = 2iIm(z)
  • conjugate(z + w) = z̄ + w̄
  • conjugate(zw) = z̄w̄

8. Modulus of a Complex Number

If z = a + ib, then:

|z| = √(a² + b²)

Properties of Modulus

  • |z| ≥ 0
  • |z| = 0 iff z = 0
  • |z₁z₂| = |z₁||z₂|
  • |z₁ / z₂| = |z₁| / |z₂|, z₂ ≠ 0
  • |z + w| ≤ |z| + |w|

9. Forms of Complex Numbers

(a) Cartesian Form

z = x + iy

(b) Polar Form

z = r(cos θ + i sin θ)

r = √(x² + y²)
tan θ = y / x

(c) Exponential Form

z = re^(iθ)

e^(iθ) = cos θ + i sin θ

10. De Moivre’s Theorem

[r(cos θ + i sin θ)]ⁿ = rⁿ(cos nθ + i sin nθ)

11. Roots of Unity

Cube roots of unity:

1, (−1 + √3i)/2, (−1 − √3i)/2

Sum = 0
Product = 1

12. Square Roots of Complex Numbers

x² = (√(a² + b²) + a) / 2
y² = (√(a² + b²) − a) / 2

Square roots of 5 + 12i are ±(3 + 2i)

13. Important Results

Distance between z₁ and z₂ = |z₁ − z₂|

If |(z − 4)/(z − 2)| = 1, then Re(z) = 3

14. Area Property

Area of triangle formed by z, iz and z + iz is (1/2)|z|²
