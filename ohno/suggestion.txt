## Approach

**Use a Python-based parser** with these libraries:
1. **TexSoup** - Parse LaTeX into a structured format
2. **Regular expressions** - Extract table data and math expressions
3. **urllib.parse** - URL-encode the LaTeX math for `data-latex` attribute

## Why 6-7/10 complexity?

**Easier parts:**
- Parsing basic LaTeX table structure (tabular environment)
- Converting `&` and `\\` to HTML cells/rows
- URL-encoding math expressions
- Simple find/replace operations

**Harder parts:**
- Handling nested LaTeX commands properly
- Preserving math mode delimiters (`$...$`, `\[...\]`, etc.)
- Supporting various table formats (tabular, tabularx, longtable)
- Edge cases in LaTeX syntax
- Maintaining whitespace and formatting

## Quick Prototype

Here's a simple Python script to get you started:

```python
import re
import urllib.parse

def latex_table_to_tiptap(latex_content):
    # Extract table content between \begin{tabular} and \end{tabular}
    table_match = re.search(r'\\begin\{tabular\}\{[^}]*\}(.*?)\\end\{tabular\}', 
                           latex_content, re.DOTALL)
    
    if not table_match:
        return None
    
    table_content = table_match.group(1).strip()
    
    # Split into rows
    rows = [row.strip() for row in table_content.split('\\\\') if row.strip()]
    
    html_rows = []
    
    for i, row in enumerate(rows):
        # Remove \hline
        row = row.replace('\\hline', '').strip()
        
        # Split by & to get cells
        cells = [cell.strip() for cell in row.split('&')]
        
        row_html = '<tr>\n'
        
        for cell in cells:
            # Check if cell contains math (between $ ... $)
            math_match = re.search(r'\$(.+?)\$', cell)
            
            if math_match:
                latex_expr = math_match.group(1)
                # URL encode the LaTeX
                encoded = urllib.parse.quote(latex_expr)
                
                tag = 'th' if i == 0 else 'td'
                row_html += f'  <{tag}><span class="tiptap-katex" data-latex="{encoded}"></span></{tag}>\n'
            else:
                tag = 'th' if i == 0 else 'td'
                row_html += f'  <{tag}>{cell}</{tag}>\n'
        
        row_html += '</tr>'
        html_rows.append(row_html)
    
    return '<table>\n' + '\n'.join(html_rows) + '\n</table>'

# Read your .tex file
with open('table.tex', 'r', encoding='utf-8') as f:
    latex_content = f.read()

# Convert
html_output = latex_table_to_tiptap(latex_content)
print(html_output)
```
