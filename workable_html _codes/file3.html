(i) Equation : -

A relation is said to be an equation if it is true for certain values of the indeterminates.

Example : -

y=x²-5 x+6=0, y=x+1=0

(ii) Roots of the equation :

The value or the set of values which satisfies the given equation are known as the roots of the equation

Example :

x²-3 x+2=0

x²-2 x-x+2=0

(x-2)(x-1)=0

x=2

x=1

1 and 2 are the roots of the equation x²-3 x+2=0 as it is satisfied by these values.

(iii) Identity : -

A relation is said to be an identity if it is true of every value of the indeterminates.

Example

(x+1)²=x²+2 x+1

This relation is an identity as it is satisfied by all the values of x.

(iv) Consistent and inconsistent

a) Consistent

System of equations is said to be consistent if it has a solution whether unique or infinite number of solution.

b) Inconsistent

If the system has no solution, then it is called inconsistent.

(v) Trivial and Non-trivial solution:

A solution in which every variable has zero value is called trivial solution i.e. x=0, y=0, z=0

A non-zero solution is called non-trivial solution.

For example

( 0,0,0 ) is a trival solution of x=2 y+z=0 and ( 1,2,3 ) is a non trivial solution.

(vi) Intersecting lines:

a) Any two geometrically intersecting

b) They are consistent and independent Example:

x+y=5

3 x-2 y=5

(vii) Parallel lines:-

a) Any two geometrically parallel lines have no solution.

b) They are inconsistent and independent. Example:

x-y=2

2 x-2 y=5 L₂

x-y=2

(viii) Coincident lines:-

a) Any two geometrically coincident lines have infinite solution.

b) They are dependent and consistent.

Example:

3 x-y=5

6 x-2 y=10

(ix) If two equations have three variables then there are infinite solutions.

Example:

x+2 y+z=5

3 x+4 y+z=7 has infinite soluti

x+2 y+z=5

2 x+4 y+2 z=7 has no solution.

(x) The system of equations

x-2 y-z=-7

2 x+y+z=0

3 x-5 y+8 z=13

has a unique solution i.e. x=-2, y=1 and z=3

(xi) Solution of simultaneous linear equations (Cramer's Rule:)

(I) Consider a system of simultaneous linear equations:-

a₁ x+b₁ y=c₁

a₂ x+b₂ y=c₂

D = | a₁ b₁ | | a₂ b₂ | = a₁b₂ - a₂b₁

D₁ = | c₁ b₁ | | c₂ b₂ | = c₁b₂ - c₂b₁

(Replacing 1st column by constant column)

D₂ = | a₁ c₁ | | a₂ c₂ | = a₁c₂ - c₂a₁

(Replacing 2nd column by constant column)

x = D₁/D and y = D₂/D

(II) Consider a system of simultaneous linear equations: -

a₁₁x₁+a₁₂x₂+a₁₃x₃=b₁

a₁₂x₁+a₂₂x₂+a₂₃x₃=b₂

a₃₁x₁+a₃₂x₂+a₃₃x₃=b₃

D = | a₁₁ a₁₂ a₁₃ | | a₂₁ a₂₂ a₂₃ | | a₃₁ a₃₂ a₃₃ |

D₁ = | b₁ a₁₂ a₁₃ | | b₂ a₂₂ a₂₃ | | b₃ a₃₂ a₃₃ |

D₂ = | a₁₁ b₁ a₁₃ | | a₂₁ b₂ a₂₃ | | a₃₁ b₃ a₃₃ |

D₃ = | a₁₁ a₁₂ b₁ | | a₂₁ a₂₂ b₂ | | a₃₁ a₁₂ b₃ |

x = D₁/D, y = D₂/D, z = D₃/D

Important Notes:-

a) If D ≠ 0 then the system has unique solution

D ≠ 0 i.e. if a₁/a₂ ≠ b₁/b₂, the system is consistent and independent

b) System has infinite solution if D = D₁ = D₂ = 0.

a₁/a₂ = b₁/b₂ = c₁/c₂, the system is dependent and consistent.

c) These equations can be written in the matrix form:

A X = B

A = | a₁₁ a₁₂ a₁₃ | | a₂₁ a₂₂ a₂₃ | | a₃₁ a₃₂ a₃₃ |

X = [ x₁ x₂ x₃ ]

B = [ b₁ b₂ b₃ ]

The matrix A is called the coefficient matrix.

X is called the matrix of unknowns.

A set of values of variables x₁, x₂, x₃ which simultaneously satisfy these three equations is called a solution.

Example:

The system of equations kx-2y=0

x+3y=0

has a unique solution then

  • a) k ≠ -2/3
  • b) k = -2/3
  • c) k = 0
  • d) k = 2/3

Solution:

| k -2 | | 1 3 | ≠ 0

3k+2 ≠ 0

k ≠ -2/3

d) If D = 0 and at least one of D₁ and D₂ ≠ 0, then system has no solution.

a₁/a₂ = b₁/b₂ ≠ c₁/c₂

The lines are parallel

Example:

kx+3y=0

x+2y=0

has no solution if k =

  • a) k = 2/3
  • b) k = 3/2
  • c) k = -1
  • d) k = 0

Solution:

| k 3 | | 1 2 | = 0

2k-3 = 0

k = 3/2
